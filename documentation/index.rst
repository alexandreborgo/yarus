YARUS
=====

Yum Apt Repositories Update Service (YARUS) est un outil de mises à jour centralisées et automatiques pour des systèmes Linux utilisant les
gestionnaires de paquets Yellowdog Updater Modified (Red Hat Enterprise Linux, CentOS, etc) et Aptitude (Debian, Ubuntu, etc).

YARUS permet de créer des dépôts miroirs locaux avec la possibilité de choisir les versions, composants et architectures à inclure dans le miroir.
Fonctionne avec les dépôts YUM et APT.
